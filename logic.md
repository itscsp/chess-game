# OOP for Chess Game
1. Board
    1. **chessBoard** - 2d matrix of prices
    2. **playerColor** - color of player which turm is now
    3. **chessBoardView** - getter which represents baord as 2d char matrix form which UI can print board

2. Pieces
    1. **FENChar** - name (representation) of Piece
    2. **color** - white or black
    3. **direction** - direction of pieces movement as array of coordinartes (x,y)
    4. White Piece Names (Upper case)
        + King (K)
        + Queen (Q)
        + Rock (R)
        + Knight (N)
        + Bishop (B)
        + Pown (P)
    5. Black Piece Names (Smaller case)
        + King (k)
        + Queen (q)
        + Rock (r)
        + Knight (n)
        + Bishop (b)
        + Pown (p)
